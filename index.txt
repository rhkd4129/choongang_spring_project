하면 devlop 브랜치 만들어진다.

git push를 하면
git push --set-upstream origin develop

develop라는 브랜치가 github에는 안만들어져 있어서 이 develop브랜치를 올리는작업


브랜치 보호정책
Look branch  개개인 함부로 푸쉬할수없다

Require a pull request before merging
반드시 pull   request 통해서만 master에 푸시할수 잇다.
git remote add origin 클론할 주소 
git push origin master  (현재 로컬 저장소에서 master브랜치로 올린다?)
